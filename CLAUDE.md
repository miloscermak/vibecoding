# Vibe Coding Workshop

Landing page pro workshop "Vibe Coding" od Miloše Čermáka a Pavla Kasíka.

## Struktura
- `index.html` – kompletní landing page (HTML + CSS + JS v jednom souboru)

## Technologie
- Čisté HTML, CSS, vanilla JavaScript
- Fonty: DM Serif Display, DM Sans, JetBrains Mono (Google Fonts)
- Žádné frameworky, žádné závislosti

## Spuštění
Stačí otevřít `index.html` v prohlížeči. Pro fonty je potřeba internet.

## Důležité
- Ceny a odkazy na eshop (SimpleShop) jsou přímo v HTML
- Registrační odkazy vedou na form.simpleshop.cz
- Termíny workshopu: 18. dubna a 23. dubna 2026
- Demo ukázky (kalkulačka, sanitka, dashboard) jsou v `<template>` tazích na konci souboru
